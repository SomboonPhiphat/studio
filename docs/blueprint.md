# **App Name**: Sawasdee Health

## Core Features:

- Demographic Data Capture: Collect and store patient demographic information: name, gender, date of birth.
- EQ-5D-5L Questionnaire: Present the EQ-5D-5L questionnaire in Thai, allowing users to select their health status across five dimensions.
- Hybrid Utility Score Calculation: Calculate the Thai hybrid model utility score based on user responses, using the Pattanaphesaj et al. (2018) coefficients.
- Visual Analog Scale (EQ-VAS): Implement a visual analog scale for patients to self-report their overall health status.
- Data Persistence: Enable local storage of patient data, including demographic information, questionnaire responses, utility score, and VAS score.
- Data Export: Allow users to export collected data in CSV format for analysis and record-keeping.
- Data Reset: Reset function to clear user inputs.

## Style Guidelines:

- Primary color: Modern Blue (#2563eb) for a professional and trustworthy feel.
- Background color: Light Slate (#f1f5f9) for a clean and calming backdrop.
- Card Background: White (#ffffff) for contrast and clarity.
- Severity Colors: Use a gradient from Green (#16a34a) to Red (#dc2626) to visually represent health severity.
- Font: 'Kanit' (sans-serif) for a modern and readable Thai interface. Note: currently only Google Fonts are supported.
- Rounded corners (16px-24px) and soft box-shadows for a premium, user-friendly feel.
- Subtle transitions on hover and selection to enhance interactivity.
